Tailwind

What is it and how can I use it?

Setup

You don't have to use the exact same setup as I did. If you use another one you have to personalize the paths.

a. Create two folders in you directory. b. Public folder with index.html and style.css file c. Src folder with tailwind.css file

Package.json

a. Make sure you are not in one of the created folders. b. Open the terminal and type: npm init -y c. This command creates a package.json file in your directory.

Tailwind

What is it and how can I use it?

Setup

You don't have to use the exact same setup as I did. If you use another one you have to personalize the paths.

a. Create two folders in you directory. b. Public folder with index.html and style.css file c. Src folder with tailwind.css file

Package.json

a. Make sure you are not in one of the created folders. b. Open the terminal and type: npm init -y c. This command creates a package.json file in your directory.

Tailwind.css

a. Open your .css file in the src folder.

c. In here you can later create your own objects for your website.

Package.json

a. Go into your .json file and scroll down to "scripts" b. Delete what's after and copy this: 

                    
                        "dev": "npx tailwindcss-cli@latest build ./src/tailwind.css -o ./public/style.css"
c. Npx is almost the same as npm. With npm you run a command and it directly executes. With npx you must run the command in the terminal after you specified what it must do. d. What this command does is that it installs the latest package of tailwindcss-cli and compiles the tailwind.css file in the src folder. After that it saves the output (-o) in the style.css file in the public folder.

Run the npx command

a. Open your terminal again and make sure you are in the same direction as your package.json file. b. Type: npm run dev. c. I use dev because I “named” my command in the package.json file dev. If you used another name, you have to replace it. d. Now open your output file. There should be round about 500 lines in it.

Tailwind

What is it and how can I use it?

1 Setup

You don't have to use the exact same setup as I did. If you use another one you have to personalize the paths.

a. Create two folders in you directory.

b. Public folder with index.html and style.css file.

c. Src folder with tailwind.css file.

2 Package.json

a. Make sure you are not in one of the created folders.

b. Open the terminal and type: npm init -y.

c. This command creates a package.json file in your directory.

3 Tailwind.css

a. Open your .css file in the src folder.

b. Copy the folowing code into the .css file:

@tailwind base;
@tailwind components;

btn {
@apply px-4 py-2 bg-blue-600 text white rounded;
}
@tailwind utilities;

c. In here you can later create your own objects for your website.

4 Package.json

a. Go into your .json file and scroll down to "scripts".

b. Delete what's after and copy this:
"dev": "npx tailwindcss-cli@latest build ./src/tailwind.css -o ./public/style.css"

c. Npx is almost the same as npm. With npm you run a command and it directly executes. With npx you must run the command in the terminal after you specified what it must do.

d. What this command does is that it installs the latest package of tailwindcss-cli and compiles the tailwind.css file in the src folder. After that it saves the output (-o) in the style.css file in the public folder.

5 Run the npm command

a. Open your terminal again and make sure you are in the same direction as your package.json file.

b. Type:
npm run dev

c. I use dev because I “named” my command in the package.json file dev. If you used another name, you have to replace it.

d. Now open your output file. There should be round about 500 lines in it.

2 Tailwind config

a. Type this command in the terminal: npx tailwindcss-cli@latest init

b. This command creates a config file. We need it because:
Every time you build to make a new css class in your tailwind.css file, it builds every tailwind class and even those you don't use. That means that our .css file is very big. To prevent that we need to create this config file.
